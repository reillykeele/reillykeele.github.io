Kart Racer (W.I.P.)

This solo endeavor serves as a concrete Unity 3D project to demonstrate my gameplay programming skills, as well as an opportunity to expand my Unity knowledge through exploration and experimentation. The focus of this project is not on design, but rather, implementing technical solutions to achieve an existing design (in this case, Mario Kart).

As this is a solo project, limiting scope, and having well-defined goals is critical to completing and releasing This project. Holding myself accountable and pushing myself to learn new things, and discover creative solutions to solve difficult problems has been difficult, but worthwhile.

While the project is not yet complete, some key areas that I've gained new experience in throughout development:

  • 3D Character Controllers
  • 3D Cameras (Cinemachine)
  • Generalizing behaviours through polymorphism
  • Designing and implementing responsive, juicy, gamepad-friendly UIs
  • Designing and programming believable racer AI behaviour
  • Shaders and particle systems (URP)
  • Utilizing event-driven patterns to reduce coupling and improve performance
  • Reducing/limiting the need for singletons (a problem in pervious team projects)

Tools and Technologies:

  • Unity 2020.3.26f1 LTS
  • C#
