Kario Kart (Prototype)

This solo endeavor serves as a concrete Unity 3D project to demonstrate my gameplay programming skills, as well as an opportunity to expand my Unity knowledge through exploration and experimentation. The focus of this project is not on design, but rather, on implementing technical solutions to achieve an existing design (in this case, Mario Kart).

As this is a solo project, limiting the scope, and having well-defined goals is critical to completing and releasing This project. Holding myself accountable, pushing myself to learn new things, and discovering creative solutions to complex problems has been challenging, but worthwhile.

While the project is not yet complete, some key areas that I've gained new experience in throughout development:

  • 3D Character Controllers
  • 3D Cameras (Cinemachine)
  • Generalizing behaviours through polymorphism
  • Designing and implementing responsive, juicy, gamepad-friendly UIs
  • Designing and programming believable racer AI behaviour
  • Surface-level knowlege of shaders and particle systems (URP)
  • Utilizing event-driven patterns to reduce coupling and improve performance
  • Reducing/limiting the need for singletons (a problem in pervious team projects)
  • Audio programming (a weak area for me)

Tools and Technologies:

  • Unity 2020.3.26f1 LTS
  • C#

More information about the project, including a Windows download of the current build, can be viewed on the Kario Kart itch.io page. You can also view the source code on GitHub.
